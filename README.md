# 🖥️ Funcionalidades

- Campo para **usuário**
- Campo para **senha**
- Botão **Entrar**
- Botão **Apagar**
- Estrutura semântica utilizando:
  - `form`
  - `fieldset`
  - `legend`
  - `label`
  - `input`
- **Estilização com CSS**
- Layout **simples e organizado**

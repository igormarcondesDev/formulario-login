# 📄 Formulário de Login

Projeto simples de um **formulário de login** desenvolvido utilizando **HTML5 e CSS3**.  
O objetivo é praticar a criação de formulários web e a organização da estrutura básica de páginas.

---

# 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- Git
- Visual Studio Code

# 🖥️ Funcionalidades

  - `fieldset`
  - `legend`
